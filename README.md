```markdown
# Kas-lalam-2026 — موقع عرض مباريات (نسخة ثابتة)

هذه نسخة ثابتة بسيطة لعرض مباريات كرة القدم باستخدام بيانات تجريبية (mock JSON).

التشغيل محليًا:
1. افتح `index.html` في المتصفح (أو شغّل عبر سيرفر محلي بسيط مثل `npx http-server` أو `python -m http.server`).
2. لربط API حقيقي لاحقًا، عدّل `app.js` لاستدعاء API خارجي بدلاً من `data/matches.json`، وستحتاج لمفتاح API.

الميزات الموجودة:
- واجهة عربية RTL
- فلاتر حسب الدوري والتاريخ
- حالات المباريات: قبل المباراة / مباشر / انتهت

إذا تريد، أقدر أضيف:
- صفحة تفاصيل لكل مباراة
- تحديث تلقائي (polling)
- ربط API خارجي مع تعليمات حول استخدام مفتاح
```
